Proyecto: reservacion_vuelos (Node.js + Express + EJS)
Estructura similar a tus proyectos anteriores (vistas EJS y rutas separadas).

Pasos para ejecutar:
1) Tener Node.js instalado.
2) En la carpeta del proyecto ejecutar:
   npm install
   node app.js
3) Abrir en el navegador http://localhost:3000

Notas:
- Almacenamiento en memoria (los datos se pierden al reiniciar la app).
- seed automático: se crean 10 vuelos la primera vez que entras a "/".
- Si quieres, puedo convertirlo a MongoDB o MySQL después.
